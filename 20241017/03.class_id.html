CSS 的選擇器

CSS 有非常多選擇器，幫助我們在撰寫樣式時，依照該情境選擇到我們要的目標。

class 選擇器

特性: 具有重複使用的特性，是使用率最高的選擇器。

超連結

特性2:

抽離相同的屬性寫成 class 重複使用，再對不一樣的屬性定義 class

class 是可以一直增加...例如 box bg-color1



id 選擇器

一份文件裡面，只允許一個獨一無二的 id

對元素(標籤)設定 id，可以用程式快速取得它。

網頁區塊名稱會用 id 來命名

對元素(標籤)設定 id，用於程式
